# Question

Every $n$-node tree has exactly $n - 1$ edges.

# Answer

True

# Explanation

Rooting the tree at any node $r$, every non-root node has exactly one edge leading to its parent. This gives a one-to-one correspondence between edges and the $n - 1$ non-root nodes.
