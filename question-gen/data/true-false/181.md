# Question

In a stable matching problem, when running the Gale-Shapley algorithm with men proposing, it is possible for men to end up with their worst valid partner.

# Answer

True

# Explanation

In G-S with men proposing, each man ends up with his best valid partner. However, if a man has only one valid partner, then his best valid partner is also his worst valid partner. So it is possible for a man to end up with his worst valid partner.
