# Question

For all positive functions $f(n)$, $g(n)$ and $h(n)$, if $f(n) = O(g(n))$ and $f(n) = \Omega(h(n))$, then $g(n) + h(n) = \Omega(f(n))$.

# Answer

True

# Explanation

Since $f(n) = \\Omega(h(n))$, there exists $c > 0$ such that $f(n) \\leq (1/c) \\cdot h(n)$ for large $n$, meaning $h(n) \\geq c \\cdot f(n)$. Therefore $g(n) + h(n) \\geq h(n) \\geq c \\cdot f(n)$, which means $g(n) + h(n) = \\Omega(f(n))$.
