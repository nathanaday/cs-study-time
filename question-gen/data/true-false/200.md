# Question

If $f = O(g)$ and $g = O(h)$, then $f = O(h)$.

# Answer

True

# Explanation

Big-O is transitive. If $f(n) \\leq c_1 \\cdot g(n)$ for large $n$, and $g(n) \\leq c_2 \\cdot h(n)$ for large $n$, then $f(n) \\leq c_1 c_2 \\cdot h(n)$ for large $n$, so $f = O(h)$.
