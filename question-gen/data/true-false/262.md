# Question

The height of an $n$-element heap is $\lceil \lg n \rceil$.

# Answer

False

# Explanation

The height of an $n$-element heap is $\lfloor \lg n \rfloor$, not $\lceil \lg n \rceil$. From the bounds $2^h \le n \le 2^{h+1} - 1$, we derive $h \le \lg n < h+1$, so $h = \lfloor \lg n \rfloor$.
