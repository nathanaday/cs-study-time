# Question

Given functions $f_1$, $f_2$, $g_1$, $g_2$ such that $f_1(n) = O(g_1(n))$ and $f_2(n) = O(g_2(n))$, it follows that $f_1(n) + f_2(n) = O(\max(g_1(n), g_2(n)))$.

# Answer

True

# Explanation

Since $f_1(n) \leq c_1 \cdot g_1(n)$ and $f_2(n) \leq c_2 \cdot g_2(n)$, we get $f_1(n) + f_2(n) \leq c_1 \cdot g_1(n) + c_2 \cdot g_2(n) \leq (c_1 + c_2) \cdot (g_1(n) + g_2(n)) \leq 2(c_1 + c_2) \cdot \max(g_1(n), g_2(n))$. Therefore $f_1 + f_2 = O(\max(g_1, g_2))$.
