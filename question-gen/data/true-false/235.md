# Question

For any tree with $n$ vertices and $m$ edges we can say that $O(m+n) = O(m)$.

# Answer

True

# Explanation

In any tree, $m = n - 1$. Therefore $m + n = (n-1) + n = 2n - 1 = O(n) = O(m+1) = O(m)$. Since $m$ and $n$ differ by exactly 1, $O(m+n) = O(m)$.
