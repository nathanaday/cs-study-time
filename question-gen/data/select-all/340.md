# Question

Which of the following recurrences solve to $O(n \log n)$?

# Choices

## a

$T(n) = 2T(n/2) + O(n)$

**Answer:** True

## b

$T(n) = T(n/2) + O(n)$

**Answer:** False

## c

$T(n) = 4T(n/2) + O(n)$

**Answer:** False

## d

$T(n) = 2T(n/2) + O(n \log n)$

**Answer:** False

# Explanation

By the Master Theorem: (a) $T(n) = 2T(n/2) + O(n)$ gives $O(n \\log n)$ (Case 2, $a=2, b=2, f=n$). (b) $T(n) = T(n/2) + O(n)$ gives $O(n)$ (geometric series). (c) $T(n) = 4T(n/2) + O(n)$ gives $O(n^2)$ (Case 1, $\\log_2 4 = 2 > 1$). (d) $T(n) = 2T(n/2) + O(n \\log n)$ gives $O(n \\log^2 n)$.
