# Question

Which is the correct order of the following functions such that $g(n)$ follows $f(n)$ in the list if and only if $f(n) = O(g(n))$? The functions are: $n^{\log n}$, $\log n!$, $\log n^n$, $\log^4 n$

# Choices

## a

$n^{\log n}$, $\log n!$, $\log n^n$, $\log^4 n$

**Answer:** False

## b

$\log n!$, $\log^4 n$, $n^{\log n}$, $\log n^n$

**Answer:** False

## c

$\log^4 n$, $\log n!$, $\log n^n$, $n^{\log n}$

**Answer:** True

## d

$n^{\log n}$, $\log n^n$, $\log^4 n$, $\log n!$

**Answer:** False

# Explanation

$\\log^4 n$ is polynomial in $\\log n$. $\\log n! = \\Theta(n \\log n)$ by Stirling's approximation. $\\log n^n = n \\log n$, so $\\log n! = \\Theta(\\log n^n)$. $n^{\\log n} = 2^{\\log^2 n}$ which grows super-polynomially. So the order is: $\\log^4 n$, $\\log n!$ (=$\\log n^n$), $n^{\\log n}$.
