# Parts

## a

For each pair of expressions (A, B), indicate whether A is $O$, $\Omega$, or $\Theta$ of B. Assume $k$ and $C$ are positive constants. (Note: log is base 2)

Row 1: $A = n^3 + \log n + n^2$, $B = Cn^3$
Row 2: $A = n^2$, $B = Cn \cdot 2^{\log n}$
Row 3: $A = 2^n \cdot 2^k$, $B = n^{2k}$

**Answer:** Row 1: $n^3 + \log n + n^2 = \Theta(n^3)$ and $Cn^3 = \Theta(n^3)$, so $A = \Theta(B)$. (O: Yes, Omega: Yes, Theta: Yes)

Row 2: $Cn \cdot 2^{\log n} = Cn \cdot n = Cn^2$, so $A = n^2 = \Theta(Cn^2)$. (O: Yes, Omega: Yes, Theta: Yes)

Row 3: $2^n \cdot 2^k = 2^{n+k}$ which is exponential, while $n^{2k}$ is polynomial. So $A = \Omega(B)$ but $A \neq O(B)$. (O: No, Omega: Yes, Theta: No)
