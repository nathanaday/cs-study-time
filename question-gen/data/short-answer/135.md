# Background

Arrange these functions under the Big-$O$ notation in increasing order of growth rate, with $g(n)$ following $f(n)$ in your list if and only if $f(n) = O(g(n))$; mention equality of growth rate if $f(n) = O(g(n))$ and also $g(n) = O(f(n))$. Use $\log(x) = \log_2(x)$.

The functions are: $2^{\log\log(n)}$, $\log\log(n)$, $n!$, $n \log\log(n^2)$, $(\log\log(n^n))$, $4^{3n}$, $n^{n\log\log(n)}$, $n^{n^2}$, $3^{4n}$

# Parts

## a

List the functions in increasing order of growth rate.

**Answer:** First simplify: $2^{\log\log(n)} = \log n$; $\log\log(n^n) = \log(n\log n) = O(\log n)$; $n\log\log(n^2) = O(n\log n)$; $4^{3n} = 64^n$; $3^{4n} = 81^n$. The increasing order is: $\log\log n < 2^{\log\log n} = \log\log(n^n)$ (both $\Theta(\log n)$) $< n\log\log(n^2)$ ($\Theta(n\log n)$) $< 4^{3n} < 3^{4n} < n! < n^{n\log\log(n)} < n^{n^2}$.
