# Background

Arrange the following functions in increasing order of growth rate with $g(n)$ following $f(n)$ in your list if and only if $f(n) = O(g(n))$:

$\log n^n$, $n^2$, $n^{\log n}$, $n \log \log n$, $2^{\log n}$, $\log^2 n$, $n^{\sqrt{2}}$

# Parts

## a

List the functions in increasing order of growth rate.

**Answer:** First simplify: $2^{\log n} = n$ (assuming base 2), and $\log n^n = n \log n$. Categorize: Logarithmic: $\log^2 n$. Polynomial: $n$, $n \log \log n$, $n \log n$, $n^{\sqrt{2}}$, $n^2$. Exponential (superpolynomial): $n^{\log n}$. Among the polynomials, $n < n \log \log n < n \log n < n^{\sqrt{2}} < n^2$. Final order: $\log^2 n$, $n$ (i.e. $2^{\log n}$), $n \log \log n$, $n \log n$ (i.e. $\log n^n$), $n^{\sqrt{2}}$, $n^2$, $n^{\log n}$.
