# Parts

## a

Rank the following functions in order from smallest asymptotic complexity to largest. (lg is log base 2): $\lg n^{10}$, $3^n$, $\lg n^{2n}$, $3n^2$, $\lg n^{\lg n}$, $10^{\lg n}$, $n^{\lg n}$, $n \lg n$

**Answer:** $\lg n^{10} < \lg n^{\lg n} < \lg n^{2n} = n \lg n < 3n^2 < 10^{\lg n} < n^{\lg n} < 3^n$.

Explanation: $\lg n^{10} = 10 \lg n$. $\lg n^{\lg n} = (\lg n)^2$. $\lg n^{2n} = 2n \lg n = \Theta(n \lg n)$. $10^{\lg n} = n^{\lg 10} \approx n^{3.32}$. $n^{\lg n} = 2^{(\lg n)^2}$ which is super-polynomial. $3^n$ is exponential.
